# CS1100 - Introduction to Java Programming

A collection of Java programs from my introductory programming course, organized by concept.

## Folder Structure

| Folder | Topic | Description |
|--------|-------|-------------|
| `01-basics/` | Basics | Hello World, display output, ASCII art pattern, countdown |
| `02-input-output/` | Input & Output | Scanner input, dollar/cents formatting, stats tables, string reading |
| `03-arithmetic/` | Arithmetic & Math | Addition, area/perimeter, sphere volume, unit conversions |
| `04-conditionals/` | Conditionals | if/else and switch — BMI, gratuity, grade converter, rock-paper-scissors |
| `05-loops/` | Loops | for/while loops — countdowns, multiplication tables, sentinel values |
| `06-strings/` | Strings | Palindrome checkers, vowel/consonant counter, lottery game |
| `07-arrays/` | Arrays | 1D and 2D arrays — initialization, searching, grids |
| `08-methods/` | Methods | Custom methods — calculators, equality checker, modular programs |
| `09-games/` | Games & Projects | Squares game, grocery store inventory manager |

## How to Run

Each file is a standalone Java program with its own `main` method. To compile and run any file:

```bash
javac FileName.java
java FileName
```
